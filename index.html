Favicon & browsers

Modern browsers support more than just ICO favicons. However, to what extent? Nobody knows. Wikipedia has a nice page about it, but how often it is updated? Across what browsers have it been tested?

I've created a test suite for various use cases of favicons, varying in size, file format, and markup. Browser tests were carried out in BrowserStack.

If you want to test any of the features yourself, proceed to the test index for all the links.

Footnotes

  1. Regular sizes are tested using ICO format
  2. Irregular sizes are tested using GIF/PNG formats, as .ICO only supports squared aspect ratio
  3. Tested in Safari 4 and above.
  4. Tested in Edge 14 and above.
  5. SVG favicons are not distorted, aspect ratio is preserved by means of cover fit (similar to background-size: cover).
  6. Supported favicons are not distorted, aspect ratio is preserved by means of contain fit (similar to background-size: contain).
  7. IE7–10 only supports ICO file types, which do not support non-square aspect ratios, so fitting cannot be tested.
